# Zero-Sum Game

- = Finite Game (有限賽局) = Loser's Game = Multi-Player Game = Status Game
- You win by avoiding errors and waiting for your opponent to make errors.
- The goal is to win the game.
- 打敗多少人？(Competition)
- Either WIN or LOSE
	- Any gain corresponds directly with an equivalent loss.

# Positive-Sum Game

- = Infinite Game (無限賽局) = Winner's Game = Single-Player Game = Wealth Game
- You win by hitting incredible/elegant shots.
- The goal is to stay and continue playing in the game.
- 幫助多少人？(Collaboration)
- WIN-WIN
